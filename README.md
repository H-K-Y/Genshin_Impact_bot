**这个项目目前正在扩展，加入更多原神相关娱乐和信息查询功能，敬请期待**

## 简介

这个插件帮助群员在QQ群内进行诸如查询资源点位等功能。

相应的，也加入了一些趣味性的功能如原神抽卡，黄历，抽签等；
